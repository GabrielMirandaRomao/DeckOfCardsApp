# DeckOfCardsApp

DeckOfCardsApp is an android application that displays a deck of cards that you can shuffled and reshuffled, get a new deck and also, play a simple and quick version of blackJack.

## Technologies Used

- Kotlin 
- Glide
- Coil
- Koin
- Jetpack Compose
- MVVM Architecture
- Retrofit and OkHttp
